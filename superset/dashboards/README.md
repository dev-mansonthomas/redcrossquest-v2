# Superset Dashboard SQL Queries

Ce dossier contient les requêtes SQL utilisées pour créer les Virtual Datasets dans Superset.

## Structure

- `admin/` — Requêtes pour les dashboards admin
- `public/` — Requêtes pour les dashboards publics (futur)

## Usage

1. Ouvrir Superset → SQL Lab
2. Coller la requête SQL du fichier souhaité
3. Exécuter pour vérifier
4. Save → Save as new virtual dataset
5. Créer un chart basé sur ce dataset
6. Ajouter le chart à un dashboard

## Dashboards disponibles

| Fichier | Description |
|---------|-------------|
| `admin/yearly_goal.sql` | Objectifs annuels cumulés par jour |
